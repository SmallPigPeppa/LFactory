# 版本更新记录

本页面用于记录仓库中的用户可见增强和行为变化。

## 2026-04-05

### WebUI 易用性与多语言

- 为 LlamaBoard 的交互控件增加了逐字段 `?` 帮助弹窗。
- 为 Zena 头像增加语言与主题快速菜单。
- WebUI 语言选项扩展为：`en`、`ro`、`hu`、`he`、`fr`、`de`、`es`、`pt`。
- 当语言键缺失时，增加自动回退到英文的逻辑，避免标签显示异常。
- 优化头像图片回退顺序，减少顶部头像缺失场景。

### 导出流程体验

- 优化 GGUF 导出交互，补充量化说明与更安全的默认建议。
- 完善高级导出选项的说明文案。

### 运行与验证说明

- 增加前端动态绑定逻辑，确保对 Gradio 动态渲染控件也能挂载帮助入口。
- 调整 UI 脚本启动路径，提升页面加载阶段脚本执行稳定性。

## 维护方式

当你合并用户可见变更时：

1. 在顶部新增日期分组。
2. 按功能域分组（如 WebUI、训练、推理、导出）。
3. 每条尽量简洁，描述行为变化而不是实现细节。
